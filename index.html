Truce

Truce is an RPC framework and API specification language powered by CUE.

Get started

Open-source MIT Licensed. GitHub v0.0.1

Machine Friendly

Automatically generate types, servers and clients, in a variety of languages, defined using a set of unified specifications.

and Human Friendly

Express human-readable APIs, which don't require Truce support in the language of your choice, in order for clients to be hand-crafted.

Powered by CUE

CUE is at the heart of Truce. It ensures validity and makes Truce expressive.

Built in Go...

With first-class support for generating types, clients and servers in the language.

... but meant for more.

The Truce specification is meant be read and interpretted by other languages. Watch this space.
